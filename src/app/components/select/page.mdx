# Select

O componente `Select` é um wrapper estilizado do `AntdSelect`, com suporte a estados de validação (`error`, `warning`, `success`) e integração simplificada com listas de opções.

---

## Quando usar

- Para permitir a seleção de **um ou mais itens** em um dropdown.  
- Quando precisa exibir mensagens visuais de status no input.  
- Para manter consistência com os outros campos de formulário da biblioteca.

---

## Exemplo básico

<CardComponents
  title="Select básico"
  code={`() => {
  const [value, setValue] = React.useState(null);
  return (
    <Select
      placeholder="Selecione uma opção"
      value={value}
      onChange={setValue}
      options={[
        { label: "Admin", value: "admin" },
        { label: "Editor", value: "editor" },
        { label: "Viewer", value: "viewer" },
      ]}
      style={{ width: 240 }}
    />
  );
}`}
/>

## API

### SelectProps

| Propriedade   | Tipo                               | Padrão          | Descrição |
|---------------|------------------------------------|-----------------|------------|
| `options`     | `SelectOption[]`                   | `[]`            | Lista de opções disponíveis. |
| `status`      | `"error" \| "warning" \| "success"`| `undefined`     | Define o estado visual do select. |
| `dataTestId`  | `string`                           | `undefined`     | Atributo para testes automatizados. |
| `style`       | `React.CSSProperties`              | `undefined`     | Estilos inline adicionais. |
| `disabled`    | `boolean`                          | `false`         | Desabilita a interação. |
| `...rest`     | `AntdSelectProps`                  | -               | Todas as demais props do `AntdSelect`. |

---

### SelectOption

| Propriedade | Tipo                      | Descrição |
|-------------|---------------------------|------------|
| `label`     | `string`                  | Texto exibido. |
| `value`     | `string \| number \| boolean` | Valor associado. |
| `disabled`  | `boolean`                 | Se `true`, desabilita a opção. |
